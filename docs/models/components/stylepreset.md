# StylePreset

Pass in a style preset to guide the image model towards a particular style.
This list of style presets is subject to change.

## Example Usage

```typescript
import { StylePreset } from "stabilityai-client-typescript/models/components";

let value: StylePreset = "isometric";
```

## Values

```typescript
"enhance" | "anime" | "photographic" | "digital-art" | "comic-book" | "fantasy-art" | "line-art" | "analog-film" | "neon-punk" | "isometric" | "low-poly" | "origami" | "modeling-compound" | "cinematic" | "3d-model" | "pixel-art" | "tile-texture"
```
